# Project Overview
This is a responsive web application that must work on both desktop and mobile devices.

# Tech Stack
- Styling: Mobile-first responsive design

# Code Style
- Use modern JavaScript (ES6+)
- Prefer functional components (if using React)
- Keep code modular and reusable
- Use clear and descriptive variable names

# UI/UX Rules
- Mobile-first design approach
- Ensure responsiveness using flexbox or grid
- Avoid fixed widths; use relative units (%, rem)
- Test on multiple screen sizes

# Workflow
- ALWAYS write tests when adding new features
- Run tests after changes
- Fix root causes, not symptoms
- Keep commits small and meaningful

# Commands
- Start server: npm run dev / python manage.py runserver
- Run tests: npm test / pytest
- Build: npm run build

# Important Rules
- ALWAYS verify UI changes visually
- ALWAYS ensure mobile compatibility
- NEVER break existing functionality
- ALWAYS reuse existing components when possible
